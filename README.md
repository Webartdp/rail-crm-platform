# Rail CRM Platform

New CRM platform for field service, workforce management, time tracking, documents and billing.

This is a clean rebuild and not a MODX extension.

## Stack

- Backend: Laravel API
- Database: PostgreSQL
- Cache and queues: Redis
- Frontend: Next.js
- Mobile: later, after API stabilization

## Main folders

- `backend/` — Laravel API
- `frontend/` — Next.js web app
- `mobile/` — future mobile app
- `docs/` — architecture and product documentation
- `docker/` — local development infrastructure
