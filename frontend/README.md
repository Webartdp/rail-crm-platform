# Frontend

This folder will contain the Next.js web application.

## Interfaces

- admin dashboard
- coordinator workspace
- employee mobile-first cabinet
- approval queue
- documents
- billing
- management reports

## Employee workflow

The employee interface must support:

- Gasfahrt
- Gasfahrt beendet
- Dienstbeginn
- Stop
- Start Dienstfahrt
- Stop Dienstfahrt

## Main principle

The frontend should be simple for field employees and powerful for management.
