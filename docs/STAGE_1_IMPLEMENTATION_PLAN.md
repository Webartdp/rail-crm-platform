# Stage 1 Implementation Plan

Prepare the first working technical foundation.

## Backend tasks

- create Laravel project in backend
- configure PostgreSQL
- configure Redis
- create auth endpoints
- create users table
- create employee profiles table
- create roles and permissions tables
- create audit log table

## Frontend tasks

- create Next.js project in frontend
- create login page
- create base layout
- create dashboard placeholder
- create employee mobile-first screen placeholder

## First API modules

- auth
- users
- employees
- roles
- permissions
- audit log

## Done means

- backend starts locally
- frontend starts locally
- database connection works
- login flow is ready
- basic roles are seeded
