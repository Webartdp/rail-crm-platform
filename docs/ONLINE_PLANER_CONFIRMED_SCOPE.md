# Online Planer Confirmed Scope

Source: public website online-planer.de, reviewed on 2026-07-06.

This file must be used to avoid hallucinations.

## Confirmed platform idea

Online Planer is positioned as a SaaS platform for railway logistics.

Confirmed areas:

- disposition
- brake test documentation
- accounting
- web application for dispatch and control center
- iOS app for field data capture
- mobile time tracking
- audit log
- German and English UI
- offline capable mobile work

## Confirmed core modules

The website describes these 9 core modules:

1. Auftragsmanagement
2. Zugfahrt & Bremsproben
3. Buchhaltung & Rechnungen
4. Safety Management & Schulungen
5. Fahrzeuge & KFZ-Inspektionen
6. Reporting & Analytics
7. Gleisanschluss & Stationen
8. Personal & Stempeluhr
9. Dokumente, Belege & OCR
