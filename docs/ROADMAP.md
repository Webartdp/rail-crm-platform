# Roadmap

## Stage 1. Foundation

Goal: create the technical and business foundation.

Tasks:

- repository structure
- Docker environment
- Laravel backend skeleton
- Next.js frontend skeleton
- PostgreSQL connection
- Redis connection
- authentication
- roles and permissions
- audit log

## Stage 2. CRM core

Goal: create the main entities.

Tasks:

- employees
- clients
- contacts
- objects
- work locations
- assignments

## Stage 3. Field operations

Goal: support employee work scenarios.

Tasks:

- start work
- stop work
- travel events
- current employee status
- location snapshots
- mobile-first employee interface

## Stage 4. Time tracking and approvals

Goal: make time data reliable and approved.

Tasks:

- timesheets
- corrections
- approval queue
- manager review
- recalculation rules

## Stage 5. Documents

Goal: generate and store operational documents.

Tasks:

- PDF reports
- acts
- templates
- signatures
- email sending

## Stage 6. Billing

Goal: connect approved work data to money.

Tasks:

- invoices
- payment statuses
- reminders
- export

## Stage 7. Management

Goal: give management a full overview.

Tasks:

- dashboard
- KPI
- map
- filters
- activity journal
