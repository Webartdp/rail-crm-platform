# Roles

## Base roles

### Super Admin

Full access to the system.

### Admin

Manages CRM data and daily operations.

### Coordinator

Controls assignments, employee schedules, work sessions and approval queue.

### Manager

Reviews reports, approves timesheets and monitors team performance.

### Employee

Uses the field interface.

Can view own assignments, start work, stop work, send travel events, confirm timesheets and sign documents.

### Accountant

Works with approved time data, invoices, payment statuses and export.

### Client Viewer

Future limited customer role for reports and selected documents.

## Permission groups

- users
- roles
- clients
- objects
- assignments
- work sessions
- timesheets
- documents
- invoices
- audit log
- dashboard

## Rule

Permissions must be checked on the backend. Frontend checks are only for user experience.
